# Design Checklist — 30 项自检

设计完成（或主题改造完成）前，逐条勾选。任何 ✗ 都必须先修才能 ship。

---

## A. 色彩对比（10 项）

- [ ] **A1. 正文文字对比 ≥ 7:1**（不只是 WCAG 4.5:1）。用 https://webaim.org/resources/contrastchecker 测。
- [ ] **A2. 辅助文字对比 ≥ 4.5:1**（次要标签、说明文字）。
- [ ] **A3. 按钮文字对比 ≥ 4.5:1**，且按钮 disabled 态对比 ≥ 3:1（仍要看得见）。
- [ ] **A4. 状态色独立**：red/green/yellow 不与主题色混用，跨主题语义恒定。
- [ ] **A5. 灰色不是纯灰**：`--ink-muted` 带 5-10% 主色 hue 偏移（Gurney unity）。
- [ ] **A6. 深色主题的白偏色**：不用 `#FFFFFF` 纯白，用 `#FFFBF0` 暖白 / `#F5F8FF` 冷白 / `#E8FFF8` 青白。
- [ ] **A7. 同 token 跨主题分别校准**：不假设一个 hex 在所有主题都 OK。
- [ ] **A8. 一屏最多一处满饱和色**：CTA 是唯一满饱和，其他降到 ≤ 60% chroma。
- [ ] **A9. 互补对比用在 CTA**：主色 + 按钮色处于色环对角，差 180°。
- [ ] **A10. token 命名是语义**：`accent-soft / accent-strong`，不是 `sky-100 / sky-700`。

## B. 形状语言（5 项）

- [ ] **B1. 0 个尖角**：所有矩形圆角 ≥ 8px，儿童产品 ≥ 12px。
- [ ] **B2. 圆角档位 ≤ 5 个**：xs / sm / md / lg / xl 五档，不要 50 种不同圆角。
- [ ] **B3. 主 CTA 有下沉式底色**：3-4px 深一档色块，按下时隐藏 + 主体下沉 — Duolingo 的灵魂。
- [ ] **B4. 卡片 elevation 最多 2 级**：paper（main）/ paper-elevated（hover/modal）。不要 4 层叠层。
- [ ] **B5. 按下态有视觉反馈**：transform: scale(0.95) 或 translateY(2px) + 阴影减弱。

## C. 字体节奏（5 项）

- [ ] **C1. 字号在 7 档斐波那契内**：12 / 14 / 16 / 20 / 24 / 32 / 48。off-scale = 业余。
- [ ] **C2. 字重 ≤ 3 档**：title 700-800 / body 500-600 / label 400-500。
- [ ] **C3. 中文 + 数字用不同字体**：中文 PingFang / 思源黑体，数字 Fredoka / Quicksand。让数字有"立体感"。
- [ ] **C4. 行高分层**：display ≤ 1.2，body 1.4-1.5。display 紧、body 透气。
- [ ] **C5. 计时 / 积分用 tabular-nums**：`font-variant-numeric: tabular-nums;` 否则数字会"跳"。

## D. 间距 + 阴影（4 项）

- [ ] **D1. 全 8pt 网格**：所有 margin / padding 是 4 的倍数（次级）或 8 的倍数（主）。
- [ ] **D2. 同组 4-8px / 子段 16-24px / 段落 32-48px**：间距等级反映内容等级。
- [ ] **D3. 阴影色随主题**：浅主题黑阴影、深主题 glow（主色 tinted）、工具主题 1px 描边。
- [ ] **D4. 阴影色补色化**：CTA 按钮阴影用主色补色（绿按钮配紫红阴影）— Gurney 1.5。

## E. 多主题区分度（6 项 — 最常违反）

- [ ] **E1. 主色 hue 两两 ≥ 90°**：用 https://oklch.com 检查。
- [ ] **E2. 温度三分**：一暖 / 一冷 / 一中性偏冷。
- [ ] **E3. 容器明度 L\* 阶梯 ≥ 25**（或深色之间 hue ≥ 30°）。
- [ ] **E4. 形状语言随主题变**：圆角 + 按钮形状 + 边框 — 三主题三套。
- [ ] **E5. 每主题有标志性装饰**：cozy 撕边 / starry 星点 / mecha 网格。装饰 = 主题指纹。
- [ ] **E6. 字重 + tracking 随主题变**：同字体不同 posture。

---

## 快速排雷指南

如果时间紧只能查 3 条：
1. **A1**（正文对比 7:1）— 90% 的"看着 cheap"靠这一条修。
2. **A4**（状态色独立）— 跨主题语义不能崩。
3. **E4**（形状随主题变）— 三主题视觉区分度差 80% 是因为这一条。

如果用户说"看着像换皮不像换主题"，问题一定在 **E4 + E5 + E6**。

如果用户说"白色看不见"或"灰色糊"，问题在 **A1 + A2 + A6 + A7**。

如果用户说"幼稚"，问题在 **A8**（饱和度堆叠）。

如果用户说"太单调"，问题在 **B3**（缺下沉立体感）+ **C3**（中数字未分离）。
