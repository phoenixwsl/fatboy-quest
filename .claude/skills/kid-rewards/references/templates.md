## 1. 三时间档定价模板

**结构**：任何时刻商店里必须同时有：
- **即时档**（≤ 1 天可达）：30-150 分
- **中期档**（3-10 天可达）：200-600 分
- **长期档**（4 周+）：1000-5000 分（做成进度条）

**为什么有效**：覆盖 ADHD 孩子的不同时间感知阈值；任何时间维度的需求都有出路。缺任何一档触发 SKILL.md / anti-patterns A3。

**示例数字**（基于 fatboy 假设日均 60-100 分）：

| 档位 | 价格区间 | 例 | 库存帽 |
|---|---|---|---|
| 即时 | 30-80 | 一颗巧克力 / 选今晚菜 / 桌面小盲盒 | 每周 ≤ 5 |
| 即时 | 100-150 | 蜜雪冰城 / 多肉小杯 / 解压玩具 | 每周 ≤ 3 |
| 中期 | 200-400 | 笔筒 / 中型解压 / 桌游小卡 | 每周 ≤ 2 |
| 中期 | 400-600 | DQ 雪糕券 / 苔藓微景观 / 中型乐高 | 每周 ≤ 1 |
| 长期 | 1000-2000 | 大型解压套装 / LED 时钟 | 季度 ≤ 1 |
| 长期 | 2000-5000 | 大乐高套装 / 全家电影院日 | 季度 ≤ 1 |

**校验法**：
- 单日峰值 ≈ 周内中等大件价格 ÷ 5（5 天可达中件）
- 单周累计 ≈ 周内最贵大件价格 ÷ 1（一周可够最贵中件）

如果数据不符，先调价格不调赚取速率（赚取速率改动会牵连所有任务平衡）。

## 2. 4 分类货架（fatboy 已选方案）

```
🌱 小花园 ／ 🪴 桌面角 ／ 🧸 玩具堆 ／ 🍦 补给站
```

每类用 chip 二级筛选，孩子默认看"全部"。

### 🌱 小花园 — 养成属性自带反通胀

- chips: `#多肉` `#小盆栽` `#食虫植物` `#培育套装` `#微景观`
- 关键吸引力：含羞草 / 捕蝇草 / 食虫植物（"会动的植物"）
- 长期勾子：食虫植物大套装（季度大件，2000+）
- 反通胀机制：活物有自然消耗（养死/养活都是反馈），不会无限堆积

### 🪴 桌面角 — 物理空间天然反通胀

- chips: `#摆件` `#灯具` `#奖杯` `#限定`
- 关键吸引力：盲盒 / 像素灯 / 自定义奖杯（按累计积分阶段刻字）
- 长期勾子：阶段奖杯系列（1000 / 5000 / 10000 累计解锁，自动刻字）
- 反通胀机制：约定家中"展示位 5 个"，新进则旧出（捐学校 / 收纳箱）

### 🧸 玩具堆 — 含乐高，最易通胀，必须设硬规则

- chips: `#乐高` `#拼搭` `#毛绒` `#解压` `#益智桌游`
- 关键吸引力：抽抽乐 / 解压玩具 / 中型乐高
- 反通胀：大件做许愿池，季度限购 1 个 1500+ 大物
- ShopManager 中加月度配额硬限制（家长上架超过配额时弹警告）

### 🍦 补给站 — 最 ADHD 友好的多巴胺通路，必须封顶

- chips: `#零食` `#饮品` `#聚餐` `#特权`
- 关键吸引力：选择权（详见 #8）—— 30-100 分一次
- 库存：每周硬性封顶 3 次甜品

## 3. 技能券库（8 张，全部"行为解锁"，不可买）

| 卡名 | 效果 | 解锁条件 |
|---|---|---|
| 守护卡 | 抵 1 次连击中断 | 连续 7 天达标 → 1 张 |
| 豁免券 | 缺勤一天不算断击 | 每周一自动 2 张（已实施） |
| 延时券 | 任务超时缓冲 5 分钟不扣分 | 一周完成 ≥3 个 3 星 → 1 张 |
| 替换券 | 今日换掉某个非必做任务 | 每周固定 1 张 |
| 暂停券 | 暂停超 3 分钟不强退 | 累计积分每涨 200 → 1 张 |
| 求助券 | 一键求助家长，零额外推送费用 | 默认无限 |
| 跳过券 | 跳过 1 个非必做任务 | 每月 1 张 |
| 神秘券 | 抽 1 次盲盒（变率奖励） | 季度任务全勤 → 1 张 |

**核心设计原则**：
- 一律不能用积分买（详见 anti-patterns B2）
- 卡片可累积，但每张有"过期"——发出 30 天后失效，避免囤积成"减负武器库"
- UI 在首页 streak 模块旁，叫"我的卡片"，与商店分离

## 4. 许愿池机制（大件做进度条而非售卖）

**结构**：
1. 孩子在商店选一件长期大件（1500+）"我想要这个"。
2. 该件从商店列表移除，进入"我的心愿池"区（首页一角常驻可见）。
3. 进度条立刻显示 **10-15% 起步**（endowed progress）。
4. 之后每次得分按设定比例（默认 50%）自动流入许愿池，剩余 50% 留作可花积分。
5. 许愿后 7 天不能改愿（commitment device），7 天后可"花 30% 已积愿池金额"撤销。
6. 进度条最后 20% 视觉拉长 + 颜色变金 + 首页置顶提醒（goal gradient）。
7. 达成时仪式：家长一起开箱 + 孩子讲一遍由来。

**为什么有效**：
- Endowed progress（开局 10%）+ goal gradient（最后 20% 加速）+ commitment device（7 天锁定）+ ritual（开箱仪式）四层心理学叠加。
- 把"购买决策"换成"达成体验"——大件不再是"花一笔钱"，是"完成一段旅程"。
- 自动流入比例（50%）保证孩子还有现金流换即时小奖，不会"为攒大件什么都换不了"导致挫败。

**校验**：流入比例不要设到 80%+，否则即时档没钱换 → motivation cliff。

## 5. 双货币：可花积分 + 终身积分

**结构**：
- **可花积分**（spendable）：动态值，兑换扣减。
- **终身积分**（lifetime）：永远只增不减。每次得分两边同步涨，兑换时只扣 spendable，lifetime 不动。

**显示位置**：首页醒目位置同时显示两个数字，标签明确：

```
可换：1230 分        总成就：4580 分（等级 3）
```

**为什么有效**：
- 解决"花掉努力感"悖论——无论怎么兑换，成就分永不归零。
- 等级 / 称号 / 皮肤等长期奖励挂在 lifetime 上，是无通胀的成就型激励。
- ADHD 孩子的延迟奖励厌恶被部分缓解——长期目标（解锁下一等级）一直在动。

**实施细节**：
- lifetime points 不允许任何方式扣减（包括 admin、debug、reset）；只允许"看不见但可累计"
- 数据库迁移时，老用户的 lifetime points = 历史总收入（PointsEntry 中所有 delta > 0 之和）

## 6. 终身积分等级模板（6 级）

| 级 | 累计积分 | 称号（mastery 导向） | 解锁奖励 |
|---|---|---|---|
| 1 | 0 | 萌新冒险家 | 默认皮肤 |
| 2 | 500 | 坚持的肥仔 | 解锁 1 个新皮肤 + 桌面角"勋章" |
| 3 | 2000 | 闯关熟练工 | 解锁 1 个新主题（cozy/starry/mecha 之一）|
| 4 | 5000 | 任务征服者 | 解锁 1 个限量蛋仔皮肤 + 自定义奖杯（刻字"5000 分达成"）|
| 5 | 10000 | 长跑者 | 解锁神秘券 / 全家定制活动 |
| 6 | 20000 | 时间的朋友 | 终极皮肤 + 个性化称号（孩子自选词） |

**关键设计**：
- 等级解锁不可花，是"成就型"奖励。
- 等级名 mastery 导向（强调过程 / 品质，不是分数 / 名次）。详见 anti-patterns C3。
- 高级别加入"自选"元素（自定义称号 / 自选皮肤），强化 SDT 自主感。
- 锁定区里展示"下一等级未解锁"的剪影提示——curiosity gap 利用。

## 7. 强制轮转 + 季度 curation

### 周轮转（自动）

- 每周一系统自动从全库存中抽取 5-7 件展示
- 上周已展示但未售的 30%-50% 进入"暂存" 1-2 周
- 算法考虑：兼顾价格档分布（保证三时间档都有代表）+ 优先未展示过的新品
- ShopItem 加字段 `rotationStatus: 'displayed' | 'shelved'`

### 季度 curation（家长操作）

- 季节交替时家长在 ShopManager 重设货架
- 旧货 30% 留下、降价 20%
- 新货 70% 上架
- 季节限定上架（春节 / 暑假 / 圣诞）
- 系统提供"换季向导"按钮，引导家长完成

**为什么有效**：
- 抗 hedonic adaptation
- 制造稀缺感（"这周不在，下周回来啦"）
- 强迫家长定期参与设计，避免"上架一次再不管"

**示例**：fatboy 实施时可在每季度首日（3/1, 6/1, 9/1, 12/1）给家长推送 Bark "换季提醒，打开 ShopManager 重设货架"。

## 8. 选择权型奖励清单（补给站 #特权 chip）

**结构**：30-100 分换一次"决策权"。零物质成本，对孩子价值高。

| 名 | 价 | 备注 |
|---|---|---|
| 今晚饭桌选 1 道菜 | 30 | 即时档 |
| 周末早餐我决定 | 50 | |
| 30 分钟自由屏幕时间 | 80 | 家长设上限 |
| 选今晚睡前故事 | 30 | |
| 周末出门去哪我选 | 100 | 中期档 |
| 选周末是否做家务 | 80 | |
| 拍一张正式照片爸爸帮我修 | 50 | |
| 临时换座位 | 50 | 在家学习时 |

**为什么有效**：
- SDT 自主感的核心 — 孩子真切感觉"我说了算"
- 反通胀 — 不消耗物质资源
- ADHD 友好 — 即时兑现，反馈快
- 不易 hedonic adapt — 每次"我选的"是新鲜决策

**实施提示**：家长可在 ShopManager 自定义本家适用的选择权清单（不同家庭文化不同）。

## 9. 见证奖励机制

**结构**：家长在 ShopManager 增加"今日见证"按钮。
- 文本框 + 一组贴纸 emoji 选择器
- 提交后生成一张"温柔时刻贴纸卡"，进入孩子端"我的贴纸墙"
- 不入积分系统、不可兑换
- 月末家长可一键导出"贴纸墙汇总"，打印贴墙上

**示例条目**：
- "5/10 — 你主动让妹妹先选玩具，我看到了。" 🌟
- "5/12 — 自己想出来用拼图教妹妹拼，妈妈很惊喜。" ✨
- "5/15 — 输了也没哭，下棋大师啊。" 🎯

**为什么有效**：
- 捕捉系统外、内在驱动的行为（看书 / 善良 / 创造）
- 不 crowd out 内在动机（无积分、无交易）
- 满足 SDT relatedness（家长在 loop 里）
- 长期身份认同（"我是温柔 / 创造性的孩子"）

**校验**：见证条目必须**具体行为**（不是"你今天真乖"），否则失去 mastery framing 价值。家长 UI 可加"具体一点"的 placeholder 提示。

## 10. 月度净存监控（家长 Dashboard）

**结构**：家长 Dashboard 增加一块"经济健康度"卡：
- 近 30 天 收入（积分） / 兑换支出 / 净存
- 近 7 天 兑换次数
- 库存周转：哪些商品 30 天没人换

**告警规则**：

| 触发条件 | 提示 | 建议动作 |
|---|---|---|
| 净存连续 3 周 > 周收入 50% | 通胀风险 | 上架新货 / 调价 / 加大件 |
| 兑换次数连续 2 周 < 1 | 商店吸引力低 | 重新审视商品是否符合孩子兴趣 |
| 某商品 60 天 0 兑换 | 滞销 | 考虑下架 / 降价 / 换包装 |
| 某商品占兑换次数 > 60% | 单一依赖 | 引入同档替代品分散注意 |

**为什么有效**：
- 把人工的"商店经营感"半自动化
- 通胀 / hedonic adaptation 等问题有数据支撑、有触发时机
- 家长不需要时刻关心，告警时再介入

**实施提示**：告警仅 Dashboard 内显示，不推送 Bark（避免家长推送疲劳）。
