# 反模式清单（kid-rewards skill 配套）

每条反模式标三件事：长什么样、为什么坏、应该怎么做。

按"严重度"（高/中/低）排序——高严重度的会破坏整套系统的健康度，中严重度容易踩坑，低严重度是细节优化。

改动前对照本表自检：A 类必查，B 类抽查，C 类有时间再优化。

---

## 严重度 A — 不要做（破坏系统健康度）

### A1. Response Cost — 从孩子已有积分里"扣分"

**长什么样**：任务超时 -10 分；犯错 -5 分；延时使用 -20 分。

**为什么坏**：临床 ADHD 研究明确显示——扣已有积分触发的羞耻反应不成比例（loss aversion × ADHD 情绪放大），长期降低系统参与度，并和孩子建立"努力 → 被惩罚"的负联结。

**应该怎么做**：
- 任何"花成本"的机制改为消耗技能券（一次性资源），不动积分余额。
- 失败 / 超时 / 犯错只用"赚不到"（不奖励），不"扣已有"。
- 文案上避免"扣"、"罚"、"减"字眼。

### A2. Crowding-Out — 给本来内在驱动的行为发积分

**长什么样**：主动看书 +10、对妹妹温柔 +5、画画 +3、自由探索 +5。

**为什么坏**：经典 SDT 实验证明——给原本有内在动机的行为发外在奖励会**取代**内在动机。奖励一旦停止，行为发生率低于从未给过奖励的对照组。

**应该怎么做**：
- 只给"有阻力"的行为发积分（作业、刷牙、收拾）。
- 内在驱动的善意 / 创造 / 探索行为用**非积分认可**：
  - 温柔时刻贴纸（妈妈手写小卡，贴墙上）
  - 作品墙（孩子的画 / 拼搭作品永久展示）
  - 口头具体表扬（"你刚才让妹妹先选玩具，这很难做到"）

### A3. 单一时间档商店

**长什么样**：商店里只有 300+ 大件，或只有 50- 小件。

**为什么坏**：
- 只有大件 → ADHD delay aversion 触发 → "永远够不到" → 放弃尝试
- 只有小件 → 无 horizon → 没有"我攒着想要那个"的长期目标 → 缺乏意义感

**应该怎么做**：商店任何时刻必须同时有即时（≤1天）/中期（3-10天）/长期（4周+）三档商品在售。详见 templates.md #1。

### A4. 大件直接挂价（错过 endowed progress + goal gradient）

**长什么样**：大乐高 4500 分，挂在商店列表里和小件并列。

**为什么坏**：心理上变成"购买决策"——要么买要么不买，没有"在路上"的体验。Endowed progress 红利、goal gradient 加速效应、达成感 vs 购买感的根本差异全部丢失。

**应该怎么做**：大件做"许愿池"——孩子许愿后该件不再显示价格，显示进度条（开局送 10-15%），所有积分自动 / 选择性流入，最后 20% 视觉冲刺。详见 templates.md #4 "许愿池机制"。

### A5. 同屏 > 7 件商品

**长什么样**：商店打开一屏列 12 件商品。

**为什么坏**：Iyengar choice overload，ADHD 孩子尤甚。决策瘫痪 → 不兑换 → 积分通胀。

**应该怎么做**：默认显示 ≤ 5 件 + 1-2 件锁定区。其余通过分类筛选 / 滚动呈现。

## 严重度 B — 容易踩坑（重要但可分阶段修）

### B1. 单一获取方式（只能"买"）

**长什么样**：所有奖励都是积分换。

**为什么坏**：单一心理通路 → 系统体感单调 → 长期失去新鲜感。

**应该怎么做**：混合"买 / 解锁 / 抽 / 见证 / 时间锁 / 许愿"六种获取方式。详见 SKILL.md 第 4 维度。

### B2. 技能券可买

**长什么样**：守护卡 100 分一张，孩子可以无限买。

**为什么坏**：把"技能"降级为"减负工具"。鼓励的不再是坚持，而是逃避。哲学上变质了。

**应该怎么做**：技能券**只能行为解锁**：连续 7 天达标得 1 张守护卡，完成 3 个 3 星任务得 1 张延时券。详见 templates.md #3 "技能券库"。

### B3. 高频变率奖励（slot machine 风险）

**长什么样**：每个任务都开盲盒；每天有抽奖券。

**为什么坏**：Variable ratio 是最强的强化形式，对 ADHD 孩子有上瘾风险。高频化会让孩子"为了抽而做"，本末倒置。

**应该怎么做**：
- 主动变率事件（盲盒 / 抽奖）每周 ≤ 2 次
- 隐性变率（lucky bonus）频率可高，因为孩子不是"决定参与"
- 变率奖励的期望值 ≥ 等价积分的 1.2 倍

### B4. 锁定区空着 / 没有"???" 商品

**长什么样**：商店里能买的都看得见，看不见的都没有。

**为什么坏**：浪费 Loewenstein 好奇心缺口效应——免费的长期勾子。

**应该怎么做**：商店任何时刻有 1-2 件 "???，累计满 X 解锁" 的剪影商品 + 模糊线索。

### B5. 货架长期不变（无轮转）

**长什么样**：商店一年商品没变，孩子打开都是熟悉的几件。

**为什么坏**：Hedonic adaptation 抗药性 → 反复刺激衰减 → 失去吸引力。

**应该怎么做**：
- 周 / 双周展示窗口 30%-50% 轮替
- 季度 curation（家长重设货架）
- 季节限定（春节灯笼、圣诞小树）

### B6. 单一货币（终身分 = 可花分）

**长什么样**：1000 分总积分，花了 200，剩 800 显示在等级旁。

**为什么坏**：每次兑换都在"花掉自己的努力"。心理上有"成就归零"的悖论。

**应该怎么做**：双货币——
- 可花积分（spendable）：动态，兑换扣减
- 终身积分（lifetime）：永远只增不减，挂载等级 / 称号 / 皮肤

两者分开显示。详见 templates.md #5。

### B7. 没有"见证奖励"

**长什么样**：所有奖励都是任务自动触发，家长不在 loop 里。

**为什么坏**：缺 SDT relatedness 维度；也错过了系统外行为（孩子在学校的善意、自发努力）的捕捉。

**应该怎么做**：家长在 ShopManager 有"今日见证"按钮——观察到一个值得记录的时刻，给个温柔时刻贴纸（不入积分），文字描述具体行为。月末汇总成"温柔时刻墙"。详见 templates.md #9。

## 严重度 C — 细节优化（系统质感）

### C1. 兑换无仪式（点击 → 扣分 → 完）

**长什么样**：商店点击兑换按钮，积分立刻扣，弹个 toast。

**为什么坏**：交易感强，叙事记忆缺失。短期 OK，长期削弱"我是个能坚持的孩子"身份认同。

**应该怎么做**：兑换瞬间弹一句具体反馈（mastery framing）："这周你完成 18 个任务……这是它换来的。"实物到家后家长摆出来 + 孩子讲一遍由来。

### C2. 失败 / 短缺时用打击型措辞

**长什么样**："积分不足"、"库存为 0"、"兑换失败"。

**为什么坏**：每个"失败"信号 ADHD 孩子都放大。

**应该怎么做**：
- "再差 80 分就能换啦" + 进度条
- "下周再上架" + 蛋仔搬箱子占位图
- "已经被换光啦，再加货中" + 排队预定按钮（可选）

### C3. 等级 / 称号偏 performance

**长什么样**："满分王"、"第一名"、"高分玩家"。

**为什么坏**：Dweck 研究：performance 导向长期削弱挑战意愿。

**应该怎么做**：mastery 导向命名："坚持者"、"探索家"、"破纪录的人"、"小连击之王（含过程）"。

### C4. 商店分类用成人词

**长什么样**：分类名"绿植 / 桌面摆件 / 文具用品"。

**为什么坏**：孩子读起来像淘宝。

**应该怎么做**：贴近孩子语言或游戏化的分类名："小花园 / 桌面角 / 玩具堆 / 补给站"。详见 templates.md #2。

### C5. 没有"选择权"型奖励

**长什么样**：所有奖励都是物品。

**为什么坏**：选择权对孩子价值高、对家长零成本，是反通胀利器。

**应该怎么做**：补给站类目里专门设 chip "#特权"——今晚菜单我选 / 周末去哪我定 / 30 分钟自由屏幕。30-100 分一次。详见 templates.md #8。

## 自检流程

每次商店改动后，对照本表过一遍：

1. 没引入 A1-A5（严重度高）？
2. B1-B7 中如果触犯了，是否有明确补偿设计 / 记录在 todo？
3. C1-C5 是细节，但累积起来影响系统质感

**如果任何 A 类被触犯，回到 SKILL.md workflow 第 1 步重新诊断；不要继续往下推进。**

如果改动通过自检但你心里还有不安，可能触发了本清单未覆盖的新模式——先记录现象，再回到 principles.md 找学术原理对照，必要时把新发现加入本清单。
